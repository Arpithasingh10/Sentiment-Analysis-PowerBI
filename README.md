# Sentiment Analysis Power BI Project

This project performs sentiment analysis on customer reviews using Python and TextBlob and visualizes insights using Power BI.

## Tools Used
- Power BI
- Python
- TextBlob
- Pandas

## Dashboard Features
- Sentiment distribution
- Product sentiment analysis
- Region sentiment analysis
- Sentiment trend over time

## Dataset
Customer reviews dataset containing product feedback.
